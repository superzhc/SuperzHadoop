# 宏和函数

> CMake 允许用户自定义宏和函数。函数和宏只有作用域上存在区别，宏没有作用域的限制。

**示例**

*函数定义*

```
function(SIMPLE REQUIRED_ARG)
    message(STATUS "Simple arguments: ${REQUIRED_ARG}, followed by ${ARGN}")
    set(${REQUIRED_ARG} "From SIMPLE" PARENT_SCOPE)
endfunction()

simple(This Foo Bar)
message("Output: ${This}")
```
