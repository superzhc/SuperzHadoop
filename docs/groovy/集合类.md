# 集合类

## List

**定义 List**

```groovy
def lst1=["a","b","c"]

// 空列表
def lst2=[]
```

**读取数据**

```groovy
lst1[0]

lst1[-1]

// 取片段
lst1[1..2]
```

**迭代**

```groovy
lst.each { println it }
```

**`collect()`**

> `collect()` 方法：对集合中的每个元素进行操作并返回一个结果集合

```groovy
lst.collect { it*2 }
```

## Map

**定义 Map**

```groovy
def map1=["name":"法外狂徒张三"]

// 空映射
def map2=[:]
```

**读取数据**

```groovy
// 方式1
map1["name"]

// 方式2
map1.name
```
