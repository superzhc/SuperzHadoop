# 数据血缘（Data Lineage）

> 数据血缘属于数据治理中的一个概念，是在数据溯源的过程中找到相关数据之间的联系，它是一个逻辑概念。

**表级血缘关系**

| name         | desc | type | note |
|--------------|------|------|-----|
| input_db     | 输入源  | T    |     |
| input_table  | 输入   | T    | 上游输入 |
| output_db    | 输出源  | T    |     |
| output_table | 输出   | T    | 下游输出 |
