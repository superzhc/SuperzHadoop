# `tail` 命令：显示文件结尾的内容

tail 命令用来查看文件末尾的数据，其基本格式如下：

```bash
[root@localhost ~]# tail [选项] 文件名
```

此命令常用的选项及含义如下表所示：

| 选项   | 含义                                                         |
| ------ | ------------------------------------------------------------ |
| `-n K` | 这里的 K 指的是行数，该选项表示输出最后 K 行，在此基础上，如果使用 -n +K，则表示从文件的第 K 行开始输出。 |
| `-c K` | 这里的 K 指的是字节数，该选项表示输出文件最后 K 个字节的内容，在此基础上，使用 -c +K 则表示从文件第 K 个字节开始输出。 |
| `-f`   | 输出文件变化后新增加的数据。                                 |
