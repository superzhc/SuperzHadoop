# 文件和目录管理

在 Linux 操作系统中，所有的文件和目录都被组织成以一个根节点 `/` 开始的倒置的树状结构。

![](images/README-20210522101359.png)

注意：目录名和文件名都是区分大小写的。

在文件系统中，有两个特殊的目录，一个是用户所在的工作目录，即当前目录，可用一个点 `.` 表示；另一个是当前目录的上一层目录，也叫父目录，用两个点 `..` 表示。
