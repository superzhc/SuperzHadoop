# 文件校验

Linux 下文件校验（文件完整性检查）命令有 `crc32`, `md5sum`, `sha1sum`, `cksum` 等等。

|  命令   | 校验算法 |        格式         |     输出      |
|:-------:|:--------:|:-------------------:|:-----------:|
|  crc32  |   CRC    |    ` $FILENAME`     | 校验值 文件名 |
| md5sum  |   md5    | `md5sum $FILENAME`  | 校验值 文件名 |
| sha1sum |   sha1   | `sha1sum $FILENAME` | 校验值 文件名 |
|  cksum  |   CRC    |  `cksum $FILENAME`  | 校验值 文件名 |

## `crc32`
