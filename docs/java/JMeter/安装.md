# 安装

> 官网地址：<https://jmeter.apache.org/>
>
> Github 地址：<https://github.com/apache/jmeter>

## 下载

> 下载地址：<https://jmeter.apache.org/download_jmeter.cgi>

![下载界面](images/安装20230724154157.png)

## 安装

下载完成压缩包后进行解压即安装完成。

## 启动

进入 `bin` 目录下，Windows 系统运行 `jmeter.bat`，Unix 系统运行 `jmeter` 。
