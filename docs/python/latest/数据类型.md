## 联合类型：`Union`

> 在 Python 中，Union 类型是一种用于表示一个变量可以是多个不同类型之一的类型注解。它是typing模块中的一个类，可以与其他类型一起使用，以指定一个变量可以接受的多个类型。

Union 类型的语法为：

```py
Union[type1, type2,...]
```

> 在 Python3.10 版本及以上，支持使用 `X|Y` 表示联合类型，即 `type1 | type2 | ...`

## `Optional`

> `Optional[x]` 是 `Union[x, None]` 简写方式
