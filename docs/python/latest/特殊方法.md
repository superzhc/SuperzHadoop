# 特殊方法

## `isinstance()`

> `isinstance()` 函数来判断一个对象是否是一个已知的类型

**语法**

```py
isinstance(object, classinfo)
```

**参数**

- object -- 实例对象。
- classinfo -- 可以是直接或间接类名、基本类型或者由它们组成的元组。

**示例**

```py
a = 2
isinstance(a,int) # True
isinstance(a,str) # False
isinstance(a,(str,int,list)) # 是元组中的一个返回 True
```
