# 日期时间

## `time`

> 此模块的实现是直接调用底层 C 库的时间函数。

在 Python 中，时间戳通常是指从特定日期（通常是 1970 年 1 月 1 日，也称为 Unix 时间戳）以来经过的秒数或毫秒数。使用标准库中的 `time` 模块来处理时间戳。

**获取当前时间的时间戳**

```py
# 当前时间的时间戳，以秒为单位
time.time()
# 当前时间戳的毫秒
time.time() * 1000
```

## `datetime`

> 用于处理日期与时间的函数和类。

- 时间部分：`time` 类表示。`time` 实例的属性包括 `hour`、`minute`、`second` 以及 `microsecond`，并且还可以包括时区信息。
- 日期部分：`date` 类来描述。`date` 类实例化后有属性 `year`，`month`，和 `day`。
- 包含日期和时间：`datetime` 类来保存由日期组件和时间组件组成的数值。

**获取当前日期**

```py
import datetime

datetime.date.today()
```

**获取当前日期时间**

```py
from datetime import datetime

datetime.now()
datetime.today()
datetime.utcnow()
```

**将时间戳转换为日期时间**

```py
import time
from datetime import datetime

timestamp = time.time()  # 以秒为单位的时间戳
dt = datetime.fromtimestamp(timestamp)
```

**将日期时间转换为时间戳**

```py
import time
from datetime import datetime

dt = datetime(2022, 9, 13, 12, 0, 0)
timestamp = int(dt.timestamp())
```

## 解析和格式化时间

使用 `strptime()` 解析时间字符串，使用 `strftime()` 来格式化时间。

```py
import time
from datetime import datetime

# 将日期时间转换为字符串
timestamp = time.time()
dt = datetime.fromtimestamp(timestamp)
formatted_time = dt.strftime("%Y-%m-%d %H:%M:%S")

# 将字符串转换为日期时间
time_string = formatted_time
dt = datetime.strptime(time_string, "%Y-%m-%d %H:%M:%S")
timestamp = int(dt.timestamp())
```

**时间格式化符号**

| 符号  | 描述                                      |
| :---: | ----------------------------------------- |
| `%y`  | 两位数的年份表示（00-99）                 |
| `%Y`  | 四位数的年份表示（000-9999）              |
| `%m`  | 月份（01-12）                             |
| `%d`  | 月内中的一天（0-31）                      |
| `%H`  | 24小时制小时数（0-23）                    |
| `%I`  | 12小时制小时数（01-12）                   |
| `%M`  | 分钟数（00-59）                           |
| `%S`  | 秒（00-59）                               |
| `%a`  | 本地简化星期名称                          |
| `%A`  | 本地完整星期名称                          |
| `%b`  | 本地简化的月份名称                        |
| `%B`  | 本地完整的月份名称                        |
| `%c`  | 本地相应的日期表示和时间表示              |
| `%j`  | 年内的一天（001-366）                     |
| `%p`  | 本地A.M.或P.M.的等价符                    |
| `%U`  | 一年中的星期数（00-53）星期天为星期的开始 |
| `%w`  | 星期（0-6），星期天为星期的开始           |
| `%W`  | 一年中的星期数（00-53）星期一为星期的开始 |
| `%x`  | 本地相应的日期表示                        |
| `%X`  | 本地相应的时间表示                        |
| `%Z`  | 当前时区的名称                            |
| `%%`  | `%` 号本身                                |
