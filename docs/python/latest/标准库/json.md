# `json`

| 方法/属性    | 描述                                   |
| ------------ | -------------------------------------- |
| `dump(...)`  | 将 Python 对象转换为 json IO 流        |
| `dumps(...)` | 将 Python 对象转换为 json 字符串       |
| `load(...)`  | 将 json 格式的文件流转换为 Python 对象 |
| `loads(...)` | 将 json 字符串转换为 Python 对象       |
