# 元组 `tuple`

> 元组（tuple）与列表类似，不同之处在于元组的元素不能修改。元组写在小括号 `()` 里，元素之间用逗号隔开。

**定义元组**

使用 `()` 创建元组

```py
empty_tuple=()
t_student=("张三",13,)
```

**方法**

| 方法           | 描述                                                |
| -------------- | --------------------------------------------------- |
| `t[i]`         | 获取指定索引位置上的元组元素                        |
| `t[start:end]` | 截取元组                                            |
| `del t`        | 元组中的元素值是不允许删除的，使用 del 删除整个元组 |
| `x in t`       | 元素是否存在                                        |
