## FAQ

### `.\venv\Scripts\Activate.ps1 : 无法加载文件 xxxx\venv\Scripts\Activate.ps1，因为在此系统上禁止运行脚本`

以管理员身份修改 PowerShell 的策略：`Set-ExecutionPolicy RemoteSigned`
