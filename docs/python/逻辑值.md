# 逻辑值

对于表达式的返回类型不是布尔类型，Python 对如下表达式判定为 False

| 表达式    | 值        |
| --------- | --------- |
| 布尔      | `False`   |
| null 类型 | `None`    |
| 整型      | `0`       |
| 浮点型    | `0.0`     |
| 空字符串  | `''` `""` |
| 空列表    | `[]`      |
| 空元组    | `()`      |
| 空字典    | `{}`      |
| 空集合    | `set()`   |
